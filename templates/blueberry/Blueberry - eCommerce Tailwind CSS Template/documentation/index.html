Dark mode

If you want to use dark mode then just place dark.css link on page head.

<link rel="stylesheet" href="assets/css/dark.css" id="dark">

RTL Customization

For rtl you can set rtl.css link on head tag 
<link rel="stylesheet" href="assets/css/rtl.css" id="add_rtl">

Box Customization

For box you can set box.css link on head tag 
<link rel="stylesheet" href="assets/css/box-1.css" id="add_box">

Color Customization

If you want to you color 4 then, you can set color-4.css link on head tag 
<link rel="stylesheet" href="assets/css/color-4.css" id="add_class">
